# Getting started

### Prerequisites

1. [Nodejs](https://nodejs.org/en/)
2. A package manager (we're using [yarn](https://yarnpkg.com) )

### Setup
Clone the repo

Run `yarn install`

# Running and development

Start the app
`yarn start`

Run tests, then poll for changes to trigger a re-run
`yarn test -o --watch`
